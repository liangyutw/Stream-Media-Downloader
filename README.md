# Stream-Media-Downloader
線上串流媒體下載、切割、快照、mp3<BR>
主要透過幾個工具所執行<BR>
<BR>
Base OS：Cent OS<BR>
後端程式：PHP<BR>
前端應用js：Jquery<BR>
框架：Laravel 5.1<BR>
<BR>
必需安裝在 OS 裡的工具<BR>
1.youtube-dl<BR>
2.Node.js<BR>
3.Socket.io<BR>
4.Express<BR>
5.FFmpeg<BR>
<br>
